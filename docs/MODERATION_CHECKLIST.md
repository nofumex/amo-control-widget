# amoMarket / Kommo Moderation Checklist

- Public widget archive built from `widget-public.zip`.
- OAuth install/callback tested on a clean account.
- Secure widget request/disposable-token validation implemented and documented.
- Privacy/security documentation prepared.
- No hardcoded customer data, user ids, Telegram ids, or amoCRM tokens.
- Uninstall/disconnect behavior tested.
- API rate limit handling verified below 7 RPS per integration.
- Webhooks are optional accelerators; historical reports rebuild from pull APIs.
- UI strings available in `ru` and `en`.
- Support contact and user-facing setup instructions prepared.
